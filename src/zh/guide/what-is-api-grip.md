# 什么是 API Grip?

API Grip 是一个现代化的 API 管理平台，帮助开发者高效地构建、测试和管理 API。

## 功能特性

- **API 测试** - 通过直观的界面快速测试您的 API
- **团队协作** - 与团队成员一起协作开发 API 项目
- **文档生成** - 轻松生成和维护 API 文档

## 架构

API Grip 由两个主要组件组成：

- **api-grip-server** - 处理 API 管理的后端服务
- **api-grip-web** - 用于与 API 交互的 Web 界面
