# 快速开始

本指南将帮助您快速启动和使用 API Grip。

## 创建账户

1. 访问 [API Grip](https://api-grip.com) 并点击 **注册**
2. 输入您的邮箱并创建密码
3. 验证您的邮箱地址

## 创建第一个项目

1. 登录后，点击 **新建项目**
2. 为项目命名
3. 开始添加您的 API 接口

## 邀请团队成员

API Grip 专为团队协作而生：

1. 进入 **项目设置** > **团队**
2. 点击 **邀请成员**
3. 输入成员的邮箱地址
