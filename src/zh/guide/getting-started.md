# 快速开始

本指南将帮助您快速启动和运行 API Grip。

## 前置要求

- Node.js 18+
- pnpm（推荐）或 npm

## 安装

### 克隆仓库

```bash
git clone https://github.com/API-Grip/api-grip-server.git
git clone https://github.com/API-Grip/api-grip-web.git
```

### 启动服务端

```bash
cd api-grip-server
pnpm install
pnpm dev
```

### 启动 Web 界面

```bash
cd api-grip-web
pnpm install
pnpm dev
```
